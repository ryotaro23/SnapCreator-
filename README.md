# ThumnailCreator

ここにプロジェクトの説明を記入してください。プロジェクトがどんな目的で、どんな機能を持つかを簡潔に説明すると良いでしょう。


## セットアップ手順

以下の手順に従ってプロジェクトをセットアップしてください。

### 依存関係のインストール

まず、必要なNode.jsのパッケージをインストールします。プロジェクトディレクトリで以下のコマンドを実行してください。

npm install

### Docker イメージのビルド

docker-compose build

### アプリケーションの起動

docker-compose up
